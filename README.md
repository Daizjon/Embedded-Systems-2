# ATmega2560 Embedded Systems

A structured collection of embedded systems projects developed on the **ATmega2560 (Arduino Mega 2560)** platform.

This repository demonstrates low-level microcontroller programming using:

- AVR Assembly
- Embedded C
- Hardware Timers
- Interrupts
- Stack Operations
- Direct Port Manipulation
- LED and Switch Control

All projects were originally developed in Atmel Studio but can be adapted to other AVR toolchains.
