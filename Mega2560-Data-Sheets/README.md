# Mega2560 Data Sheets

This folder contains reference manuals and datasheets for the Elegoo Mega 2560 board and included hardware components.

Included:
- Mega 2560 board documentation
- Breadboard and power module manuals
- Resistors, capacitors, diodes, and transistor datasheets
- RGB LED specifications

These documents were used as hardware references during development of the assembly and Arduino projects in this repository.
